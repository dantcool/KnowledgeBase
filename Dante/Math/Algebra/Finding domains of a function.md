Linear functions such as f(x) = 2x-7 the domain is all real numbers 

Quadratic functions such as f(x)=X^2+3x-5 the domain is all real number

polynomial functions such as f(x)=2x^3 -5x^2 +7x-3 the domain is all real numbers 

if there are no fractions or square roots the domain is all real numbers 

Rational fractions such as $f(x) = \frac{5}{x-2}$ 
x can be anything but cannot let the denominator into being 0 

x-2 cannot equal 0
x cannot be 2
